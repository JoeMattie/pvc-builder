# src/examples ‚Äî bundled sample designs

Sample designs offered from the project list (planfile ¬ß7). **Rule: generic subjects only ‚Äî no
creature-specific identifiers in generator code or geometry ids.**

## Files

| File | Responsibility | Key exports |
|---|---|---|
| `generators.ts` (78) | Hand-built designs via a `build(name, nodes, edges, size)` helper | `cubeFrame()` (1m cube, 8 nodes/12 members, corner conflicts), `articulatedArm()` (3 links + 2 wrapped joints) |
| `index.ts` (34) | The `EXAMPLES` registry | `Example` interface, `EXAMPLES` (4: `articulated-arm`, `cube-frame`, `trex-rigid`, `trex-pivots`) ‚Äî each `load()` returns a `Design` |
| `trex-rigid.json` | Baked 262-node / 541-pipe quad wireframe, **`schemaVersion:1`** (exercises the migration chain on load), all-rigid | ‚Äî |
| `trex-pivots.json` | Same wireframe at **`schemaVersion:6`** with a `free` ball HUB at every node (820 pairwise records, drawn as one ball per node) ‚Äî every intersection is a universal pivot | ‚Äî |

## Depends on
`../schema` (`createEmptyDesign`, `migrateToLatest`).

## Read before editing
- **"No creature identifiers" rule**: `generators.ts` and the JSON's structural ids (`n0`, `m0`)
  are clean, BUT the trex example's human-facing `name`/`description` say "T-rex". If tightening
  the rule, that display metadata is the only place it appears ‚Äî generator source stays generic.
- **`trex-rigid.json` is `schemaVersion:1`** on purpose ‚Äî it exercises the full migration chain
  on every load. Keep it valid against v1. `trex-pivots.json` is v6 (its `free` joints can't
  exist in v1). Both are generated by `scripts/gen-trex.mjs` from `~/Downloads/trexfinalfrfr.stl`
  (not in the repo): vertices are only de-duplicated (NOT decimated ‚Äî all 262 kept), and a
  **tris‚Üíquads** pass drops flat-face triangle diagonals so faces read as quads (541 pipes). The
  render-layer member caps (`MAX_*_MEMBERS` in `ui/scene`) were raised to 800 to fit it.
- New examples must `load()` a Design that passes `designSchema` (loaded content is migrated).

_Update this file when you add an example or touch the no-creature-identifiers boundary._
